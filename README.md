###  Getting Started

### Installing

* Look for chertokdmitry/braingames in packagist. 
* Install in Linux via composer: composer global require chertokdmitry/braingames.
* Now run by command: brain-games 

```

### Prerequisites

* PHP 5.3 and higher
* Composer
